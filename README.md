# 🎬 Smart Recommendation System

A hybrid movie recommendation system that combines collaborative filtering and content-based filtering to provide personalized recommendations.

---

## 🚀 Features

- Hybrid recommendation (Collaborative + Content-Based)
- Uses cosine similarity for user and item similarity
- TF-IDF for content-based filtering
- Cold-start handling using popularity-based recommendations
- Evaluation using Precision, Recall, and F1-score
- REST API built with Flask
- Interactive UI built with Streamlit

---

## 🧠 Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Flask (API)
- Streamlit (Frontend)

---

## 📊 How It Works

1. Data preprocessing and merging
2. User-item matrix creation
3. Collaborative filtering using cosine similarity
4. Content-based filtering using TF-IDF
5. Hybrid recommendation using weighted approach
6. Evaluation using precision, recall, and F1-score
